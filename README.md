# liri_node_app
Language Interpretation and Recognition Interface

## Built With

Node.js

Node APIs:
* dotenv
* node-spotify-api
* request
* fs
* geocoder

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [npm](https://www.npmjs.com)

### Installing

1. Clone this repo down to your local machine.
2. `npm install`

### Running

1. `node liri.js <command> [title/location]`

### Commands Recognized

- [x] spotify-this-song
- [x] movie-this
- [x] do-what-it-says
- [x] geocode
